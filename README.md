# Smart Cane – Assistive Navigation System

## Overview
Smart Cane is an embedded system designed to help visually impaired users detect obstacles using ultrasonic sensing and vibration feedback.

## Features
- Real-time obstacle detection
- Distance-based vibration intensity
- Tinkercad simulation for validation
- Arduino-based embedded implementation

## System Design
Ultrasonic Sensor → Arduino → Vibration Motor

## Simulation
Tinkercad used to validate logic and wiring before hardware implementation.

## Files
- firmware/ → Arduino code
- simulation/ → Tinkercad files + link
- images/ → circuit diagrams and screenshots

## Design Decisions
- Used ultrasonic sensor for low-cost distance detection
- Vibration motor chosen for non-visual feedback
- Simulation used to validate logic before hardware testing
